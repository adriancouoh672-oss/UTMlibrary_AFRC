Especificaciones del Proyecto: UTMLibrary
Objetivo: Crear un programa de consola en .NET que realice lo siguiente:

1. Estructura del Proyecto
Contener dos capas: Core y Application.

2. Capa Core (Clases)
Crear dos clases: Libro y Autor.

Clase Libro:

Propiedades: Nombre, ISBN, Género, Año de publicación, Editorial, Autores.

Clase Autor:

Propiedades: Nombre, País de origen, Año de nacimiento.

3. Capa de Aplicación (Casos de Uso)
Implementar los siguientes casos de uso:

Obtener todos los libros de sci-fi.

Obtener todos los libros que hayan sido escritos por Octavio Paz.

Obtener todos los escritores colombianos.

Obtener a todos los autores que escriban novelas románticas.

Crear un caso de uso que permita agregar un libro al sistema.

4. Entrega
Subir el código a un repositorio de GitHub con el nombre: UTMLibrary_<iniciales>
